Browser Extensions

Highlight text anywhere on the web and instantly send it to CardifyAI to generate flashcards.

Premium feature
Browser extensions are available for Premium and Professional users (and admins). You’re all set — just pick your browser below.
Chrome
Recommended

Install the CardifyAI extension in Google Chrome or any other Chromium-based browser (Brave, Edge, Arc, etc.) and generate cards directly from web pages.

  • Highlight text on any site
  • Click the CardifyAI icon in your toolbar
  • Cards appear automatically on your CardifyAI dashboard
Install for Chrome
After installation, pin the extension to your toolbar for quick access.
Microsoft Edge
Chromium-compatible

Use CardifyAI in Microsoft Edge with the same extension experience you get in Chrome.

  • Highlight text in PDFs or web articles
  • Right-click & choose “Generate flashcards with CardifyAI”
  • We’ll open your dashboard with your new deck ready to go
Install for Edge
If prompted, allow extensions from other stores to use the Chrome listing.

How the extension works

The CardifyAI extension is designed to be as simple as possible:

  1. Install the extension for your browser above.
  2. Make sure you’re logged into CardifyAI in a tab.
  3. Highlight text from lectures, articles, or PDFs in your browser.
  4. Click the CardifyAI extension icon or use the context menu option.
  5. We open your CardifyAI dashboard with your new cards ready to review and export.

For a step-by-step visual walkthrough (with GIFs), visit the About & Tutorial page.

Troubleshooting tips
  • Make sure you’re logged into CardifyAI with the same browser profile where the extension is installed.
  • If nothing happens when you click the icon, check that the extension is pinned and enabled in your browser’s extensions menu.
  • If you recently upgraded to Premium or Professional, refresh your CardifyAI dashboard so the extension can verify your plan.
  • Still stuck? You’ll be able to contact support from the About page once social links are added.
